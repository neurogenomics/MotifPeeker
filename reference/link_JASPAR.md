# Get JASPAR link for motifs

Get JASPAR link for motifs

## Usage

``` r
link_JASPAR(motif_id, download = FALSE)
```

## Arguments

- motif_id:

  A character string specifying the JASPAR motif ID.

- download:

  A logical specifying whether to return a download link or an HTML
  embeddable matrix profile link. (default = FALSE)

## Value

A character string containing the JASPAR motif link.

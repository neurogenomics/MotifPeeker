# Apply [`normalizePath`](https://rdrr.io/r/base/normalizePath.html) to a list of paths

Apply [`normalizePath`](https://rdrr.io/r/base/normalizePath.html) to a
list of paths

## Usage

``` r
normalise_paths(path_list)
```

## Arguments

- path_list:

  A list of paths.

## Value

A list of normalised paths or the input as is if contents are not a
character.
